Mini-Project — Task List App
Project Name
week-2-task-list-app
Goal
Build a simple Task List App using only:
• HTML

• CSS

• JavaScript

No React.
Required Features
The app must include:

1. Add a new task

2. Delete a task

3. Mark a task as completed

4. Filter tasks:

5. active

6. all

7. completed

8. Save tasks to localStorage

9. Load tasks after page refresh

10. Show an empty message if there are no tasks

11. Basic responsive layout



Optional Features
Do these only after the required features are complete:

• edit task text

• search tasks

• task creation date

• clear completed tasks

• dark mode
Suggested Project Structure
week-2-task-list-app/ │ ├── index.html ├── style.css └── script.js
Suggested JavaScript Functions
Try to create these functions:
addTask()
deleteTask()
toggleTask()
renderTasks()
saveTasks()
loadTasks()
filterTasks()
Useful explanation:

• render means show data on the page;

• toggle means switch between two states;

• save means store data;

• load means get saved data back.
